# Student Dropout Prediction Using Machine Learning

## Overview

This project predicts whether a student is likely to drop out using machine learning classification algorithms.

The project includes:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Feature Selection
* Multiple ML Models
* Hyperparameter Tuning
* Model Saving



# Objective

To develop an accurate machine learning model that predicts student dropout risk using academic and behavioral factors.



# Dataset

Source: Kaggle

Dataset Link:
https://www.kaggle.com/datasets/meharshanali/student-dropout-prediction-dataset



# Features Used

* GPA
* Attendance
* Study Hours
* Stress Level
* Sleep Hours
* Academic Performance
* Family Income
* Internet Access
* Extracurricular Activities

Target Variable:

* Dropout (Yes / No)


# Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib


# Machine Learning Models Used

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier
4. K-Nearest Neighbors (KNN)
5. Gradient Boosting Classifier
6. Support Vector Machine (SVM)


# Model Evaluation Metrics

* Accuracy Score
* Precision
* Recall
* F1-Score
* Confusion Matrix
* ROC Curve

# Best Performing Model

Random Forest Classifier achieved the highest accuracy among all models.
